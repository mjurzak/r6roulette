# R6 Roulette

A Rainbow Six Siege operator randomizer. Pick a side, ban operators, and spin to randomly draw your squad. Supports teammate assignments and keeps a history of past spins.

**Live site:** [mjurzak.github.io/r6roulette](http://mjurzak.github.io/r6roulette/)

## Tech Stack

Next.js 16, React 19, TypeScript, Tailwind CSS. Statically exported and deployed to GitHub Pages.

## Getting Started

```bash
npm install
npm run dev
```

Operator data is generated by a Python Selenium scraper. Run `python backend/init_json.py` before building if the `public/` JSON files are missing.

```bash
npm run build
```
